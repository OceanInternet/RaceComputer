# Arduino Race Computer
Arduino based dinghy race computer using 9DOF+GPS with csv logging for Raceqs.

Will log the following:

* Latitude/Longitude
* Heading
* Pitch
* Roll

Will display the following:

* Heading
* Roll
* Heel
* Lift/Head

## Hardware

* Arduino Nano v3
* MPU-9255 (9DOF)
* GY-NEO6MV2 (GPS)
* HM-10 BLE 4.0 (Bluetooth)
* LCD Display
